## Style Guidelines

**Language:**
- Elegant and sophisticated
- "Refined", "elegant", "seamlessly", "effortlessly"
- Focus on integration (indoor/outdoor)
- Lifestyle benefits

**Structure:**
- Always start with bold property name | price
- "EXCLUSIVE TO LOCAL REAL ESTATE! âœ¨"
- 2-3 descriptive paragraphs
- Bullet points with emojis
- Agent contact
- Branded tagline
- "Connecting. You."
- 2-3 location hashtags (not generic ones)

**What to Highlight:**
- Views (sea, mountain, golf course)
- Indoor/outdoor flow
- Entertainment spaces
- Finishes and quality
- Layout and space
- Estate amenities
- Pet-friendly

**Avoid:**
- Generic real estate jargon
- Too many exclamation marks
- Overly salesy language
- Long hashtag lists
- Generic hashtags like #RealEstate #Property
